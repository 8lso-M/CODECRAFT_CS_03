# Password Strength Checker

A simple, lightweight Python tool to evaluate the strength of your passwords based on industry standards. Perfect for quick security audits directly from your Linux terminal.

## Features

* **Fast Evaluation:** Get instant feedback on your password strength.
* **Security Criteria:** Checks for:
    * Minimum length (8+ characters)
    * Uppercase letters
    * Numbers
    * Special characters
* **Lightweight:** No external dependencies required.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/8lso-M/CODECRAFT_CS_03.git
    cd CODECRAFT_CS_03
    ```

2.  **Make it executable (Optional):**
    ```bash
    chmod +x checker.py
    ```

## Usage

Run the script using Python 3:

```bash
python3 checker.py
```

Follow the on-screen prompt to enter your password, and the script will return its strength rating and feedback.

## Requirements

* **Python 3.x**
* **OS:** Linux (though it will work on macOS and Windows as well)
